Gameplay Features
 Menu, can go to level and quit window, will return to menu after game over
 - Can go to level
 - Can quit
 - Will return to menu after game over

 In-game
 - The player, the tank shoots at enemies as they come towards you
 - The enemies, the dogs will move towards you, moving and spawning faster each time
 - You have 3 lives
 - You will get one point per enemy killed


Controls for menu:
- W to move up
- S to move down
- Spacebar to select an option

Controls in-game
- A to rotate left
- D to rotate right
- Spacebar to shoot, can be held
